# 🗓️ Day 1: Time Complexity Analysis + Array Basics

## ✅ Concepts Covered:
- What is Time Complexity?
- Big O Notation: O(1), O(n), O(n²), O(log n), O(n log n)
- How to analyze Java code for time complexity
- Basics of Arrays in Java (traversal, max/min, reversal)

## 📌 Code Files:
- `TimeComplexityExamples.java` – Examples of O(1), O(n), O(n²)
- `PrintArray.java` – Print all array elements
- `MaxMinArray.java` – Find maximum and minimum
- `ReverseArray.java` – Reverse an array

## 💡 Notes:
- Always try to reduce time complexity by avoiding nested loops.
- Big O only gives the **upper bound** – it describes the worst-case scenario.
- Space complexity matters when using extra arrays, recursion, etc.
